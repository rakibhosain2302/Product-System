## 🚀 About This Project

This is a **Full Stack Product Management System** built using:

- Laravel (REST API Backend)
- React.js (Frontend)
- MySQL Database

### 🔥 Main Feature:
👉 English input দিলে automatically Bangla translation হয়ে যায়  
👉 Product name, category, price সহ full CRUD system

---

## ⚡ Features

- 🌐 English → Bangla Auto Translation (API based)
- 📝 Add Product (EN + BN support)
- 📦 Product Listing
- ❌ Delete Product
- ⚡ Debounced API calls (optimized performance)
- 🔗 Laravel REST API
- ⚛️ React frontend (Vite)

---

## 🛠️ Tech Stack
